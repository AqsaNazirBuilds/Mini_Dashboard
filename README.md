
# Mini Admin Dashboard

A clean and responsive **Admin Dashboard** built with **HTML, CSS, and JavaScript**.  
Includes a sidebar, dynamic content sections, cards, charts, theme toggle, and a profile dropdown menu.

---

## Features

- **Sidebar Navigation:** Dashboard, Users, Reports, Settings  
- **Top Bar:** Page title, theme toggle button, profile avatar with dropdown  
- **Cards & Charts:** Overview of key metrics like users, sessions, revenue, etc.  
- **Users Table:** Editable and deletable user entries  
- **Responsive Design:** Works on desktop, tablet, and mobile  
- **Theme Toggle:** Switch between light and dark mode  
- **Profile Dropdown:** Profile, Settings, Logout options (with clickable actions)  

---

## Technologies Used

- **HTML5**  
- **CSS3 (Flexbox & Grid)**  
- **JavaScript (ES6+)**  
- **Chart.js** for dynamic charts  

---

## Installation / Usage

1. Clone the repository:  
   ```bash
   git clone https://github.com/AqsaNazirBuilds/Mini_Dashboard.git
````

2. Navigate to the project folder:

   ```bash
   cd mini-admin-dashboard
   ```
3. Open `index.html` in your browser to view the dashboard.
